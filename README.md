# Casual Anatomy: Risk Scan

Interactive client demo: a **risk scan by 1–3 photos** (runs **locally in the browser**).

## Demo
Open: https://akemi-aiai.github.io/casual-anatomy-risk-scan/

## What it does
- Upload **1–3 photos**
- Shows a **20-second scan**
- Produces:
  - Risk level
  - Quality checklist
  - Suggested seller messages (RU + 中文)

## Privacy
Photos are processed **locally** in the browser.  

## Notes
- HEIC support depends on the browser. If a photo doesn’t load — export it as JPG/PNG.
- This is **not AI** and **not a guarantee**. It’s a helper based on what’s visible in photos.
